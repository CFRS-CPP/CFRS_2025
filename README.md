# CFRS_2025
Futebol de robôs por simulação SNCT2025 IFS campus Propriá

Este repositório visa servir de referência para o Campeonato de Futebol de Robôs por Simulação do Instituto
Federal de Educação, Ciência e Tecnologia de Sergipe, Campus Propriá, a ser realizado nas atividades locais
da Semana Nacional de Ciência e Tecnologia 2025.
