# Como usar a o código base:


* Depois de instalar o simulador, troque a pasta que está instalada em **C:\Strategy** pela pasta **"Strategy"** disponibilizada aqui contendo o código base.

* Modifique no código base a parte da estratégia criando assim a seu próprio time.

* **CUIDADO:** sempre salve uma cópia de segurança da estratégia antes de alterá-la. Assim, caso ocorra algum erro, fica mais fácil de desfazer.

* Existe um código base para o time azul e um para o time amarelo. **NÃO** troque os códigos dos times.

* Quando for programar o time **amarelo** </span> só envie comandos para os robôs **y1, y2, y3, y4 e y5**.
* Quando for programar o time **azul** envie comandos apenas para os robôs **b1, b2, b3, b4 e b5**.

Montem seus times e divirtam-se!
